# 🚗 GPLX 600 - Ứng dụng Ôn Thi Bằng Lái Xe

Một ứng dụng web tĩnh hỗ trợ người dùng ôn tập và thi thử 600 câu hỏi sát hạch lý thuyết lái xe. Dự án được tối ưu hóa giao diện trực quan, dễ sử dụng trên cả máy tính và điện thoại.

## ✨ Chức năng chính

* **📚 Học theo chủ đề:** Phân loại câu hỏi theo các hạng mục (Khái niệm, Quy tắc, Biển báo, Sa hình...).
* **🚨 Chú ý Điểm liệt:** Tổng hợp và highlight các câu hỏi điểm liệt quan trọng cần ghi nhớ.
* **🔍 Tìm kiếm thông minh:** Tìm kiếm câu hỏi nhanh chóng bằng từ khóa.
* **📝 Chế độ thi thử:** Random câu hỏi để người dùng làm quen với áp lực phòng thi (nếu có tích hợp).
* **🌙 Giao diện thân thiện:** Hỗ trợ hiển thị rõ ràng, tối ưu trải nghiệm học tập (UI/UX).

## 🛠 Công nghệ sử dụng

Dự án được xây dựng theo kiến trúc Frontend tĩnh, rất phù hợp để triển khai nhanh chóng:
* **HTML5 & CSS3:** Cấu trúc và định dạng giao diện chuẩn.
* **JavaScript (ES6+):** Xử lý logic ứng dụng.
* **React 18 (qua CDN):** Quản lý component, state và render UI mà không cần setup môi trường Node.js phức tạp.

## 🚀 Hướng dẫn cài đặt và sử dụng

Vì đây là một ứng dụng tĩnh hoàn toàn, bạn không cần cài đặt các dependency phức tạp.
